This project presents an interactive Amazon Sales Dashboard built using Power BI.

## Features

* Total Sales, Orders and Quantity
* Category-wise Sales Analysis
* Monthly Sales Trend
* State-wise Performance
* Interactive Filters

## Tools Used

* Power BI
* Excel
* Data Visualization
